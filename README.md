agent三种经典范式
1. ReAct：不断重复Thought -> Action -> Observation循环，不断将将新的observation追加到历史记录中，直到它认为在Thought中找到了答案。观察Observation、行动action。2022年提出
2. Plan-and-Solve：先做好计划Plan，再按计划逐步进行Solve。2023年提出
3. Reflection：2023年提出
